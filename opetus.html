Tekstin värien muokkaaminen

Tekstin väriä voi vaihtaa color-attribuutin avulla.

Mikäli sivun tyylittelyssä käytetään CSS:ää, voidaan color-attribuutille antaa arvoja kolmella eri tavalla:

  • Heksadesimaaliarvolla (HEX-arvolla) - #FFF000
  • Värin nimellä - yellow
  • RGB-arvolla - rgb(23, 255, 4)

Värin määrittelytyylin valinta riippuu täysin tilanteesta. Mikäli halutaan äkkiä jotain päävärejä sivulle, on järkevintä käyttää värin nimeä attribuutissa, mutta mikäli halutaan tarkempaa kontrollia väreissä, HEX-arvot ovat paras ratkaisu. Jos taas halutaan määritellä värejä dynaamisesti, esim. Javascriptin avulla, on helpointa käyttää RGB-arvoja. RGB-arvot muodostuvat kolmesta pääväristä: punaisesta (Red), vihreästä (Green) ja sinisestä (Blue). Jokainen väri saa arvon väliltä 0 ... 255 ja tämä arvo kertoo, miten paljon kyseistä väriä halutaan.

Esimerkki:

body { color: red; }

h2 { color: #FF33CC; }

#linkit { color: rgb(23, 128, 255); }

Tekstin sisennys

Tekstin aloitusta voidaan sisentää CSS:n avulla käyttämällä text-indent -attribuuttia.

Attribuutille voi antaa numeerisen arvon jollain seuraavista tavoista: px, em, %, cm, pt.

Esimerkki:

h1 { text-indent: 20px; }

p { text-indent: 3em; }

.sisennetty { text-indent: 10%; }

Tekstin koristelu

Tekstin koristelulla tarkoitan tässä tapauksessa tekstin mahdollisia alle-, yli-, tai päälleviivauksia.

Tekstiä voi koristella text-decoration -attribuutin avulla. Se saa arvokseen jonkun arvoista: none, underline, line-through tai overline.

HUOM! Tekstiä ei kannata turhaan alleviivata, sillä alleviivattu teksti tarkoittaa usein linkkiä!

Esimerkki:

.alleviivattu { text-decoration: underline; }

p { text-decoration: line-through; }

Rivin korkeuden määrittäminen

CSS:n avulla voi muokata tekstin rivikorkeutta asettamalla line-height -attribuutille jonkun arvon.

Arvon asettaminen tapahtuu paljolti samalla tavalla, kuin sisennyksen asettaminenkin

HUOM! Rivikorkeuden kasvattaminen EI kasvata fonttikokoa, pelkästään rivin vaatimaa tilaa!

Esimerkki:

p { line-height: 25px; }

Tekstin asemointi vaakatasossa

Tekstiä voidaan asemoida vaakasuunnassa muokkaamalla elementin text-align -attribuuttia.

Mahdollisia arvoja ovat: left, center, right ja justify

Left ja right tasaavat tekstin vasempaan tai oikeaan laitaan, center keskittää tekstin ja justify tasaa tekstin kumpaankin reunaan.

Esimerkki:

p { text-align: left; }

.tasattu { text-align: justify; }

Kokeile itse!

Alla olevalla lomakkeella voit testata edellä opetettuja tekstin muokkaukseen liittyviä asioita. Valitsemalla eri arvoja, valikon alapuolella oleva lomake päivittyy automaattisesti.
